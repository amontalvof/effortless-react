# Overview

`useRenderCount` is a hook that shows you how many times a component is rendered. This hook is great for debugging.

### useRenderCount

```ts
const useRenderCount = () => {
    const count = useRef(1) as React.MutableRefObject<number>;
    useEffect(() => {
        count.current++;
        return;
    });
    return count.current;
};
```

### Usage

```tsx
const RenderCountComponent = () => {
    const [boolean, toggle] = useToggle(false);

    const renderCount = useRenderCount();

    return (
        <StyledContainer>
            <h1>{boolean.toString()}</h1>
            <h1>render: {renderCount}</h1>
            <StyledButton className="btn" onClick={toggle}>
                Toggle
            </StyledButton>
        </StyledContainer>
    );
};
```
